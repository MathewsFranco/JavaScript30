Type

Description

Syntax

Regular

Regular message on console

console.log(`Message`)

Interpolated

That is how you insert JS variables into some string for example

console.log(`Message ${here goes your JS stuff}`)

Styled

You can add some style directly on console

console.log( "%c Message", "font-size:20px; background:red; text-shadow: 10px 10px 0 pink" );

Warning

Log's a message in a warning template

console.warn("Warning")

Error

Log's a message in a error template

console.error(`Error`)

Info

Log's a message in a info template

console.info(`Info`)

Testing

Test the condition that you insert and logs a message in cases that the condition in not true

console.assert(condition, "Message in cases that the condition is false");

clearing

Clear all previous console messages (please don't)

console.clear()

Viewing DOM Elements

Shows you a DOM element

console.log(p) or console.dir(p)

Grouping

Arrange a groups of logs into a dropdown

console.groupCollapsed(bart); console.log(`This is bart`); console.log(`bart is 7 years old`); console.log(`bart is 49 dog years old`); console.groupEnd(bart);

counting

Tracks and log how many times that message was logged

console.count(`Message`)

timing

Tracks and log the time that any passed inserted action took

console.time(`counting time`); Any function/fetch/whatever goes here console.timeEnd(`counting time`);

All this info are detailed at MDN, please visit!
